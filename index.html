操作帮助

W / A / S / D
以游客相机为身体，在水平面内前后左右移动。
鼠标左键拖动
原地转头，调整观看方向，不绕点旋转。
鼠标右键点击
让屏幕中心准星水平对准点击位置。
Q / E
下降 / 上升，给用户手动调 z 轴高度。
Z / C / X
画面逆时针 / 顺时针旋转，X 回正。
F
保存当前视角照片；明信片风格会额外写入标题和坐标。
导览模式
系统按路线自动移动，可暂停、切站、调速。
导出评估
保存一次浏览会话的模式切换、POI 点击、拍照和录制记录。
